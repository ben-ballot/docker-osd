### Welcome to Vagrant lab ###

1. Modify IPs in servers_config.rb so it doesn't conflict with other vboxnet
 already declared on the host

2. Depending on the flavor (centos6, centos7, etc ...), use :

    make <distrib>_up

 i.e :
    make centos6_up

3. Enjoy !
